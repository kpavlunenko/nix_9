Работа с базой переделана через репозитории. Для отборов по компании или контрагенту используються предикаты

На сайте реализована возможность создания:
- компаний
- контрагентов
- договоров между компанией и контрагентом
- направлений деятельности
---------
При создании направления деятельности можно указать какие компании будут его использовать;
При удалении компании удаляються связи с направлениями деятельности, сами направления остаються (и наоборот);
Компанию, контрагента можно удалить только при условии что у них нету договоров.
---------------
Таблица в MySql: web_crud;
Создание таблиц происходит при запуске;
Для заполнения базы нужно использовать скрипт: data.sql;
