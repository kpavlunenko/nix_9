На детальной информации пользователя доступен список его счетов, доступно создание нового счета и скачивание выписки по счету;
На детальной информации счета отображаеться баланс по счету и доступен переход на список опецаий, где можно создавать новые операции по счету;
Если при внесении затрат по счету на балансу не достаточно средств, программа выдаст ошибку;
При удалении пользователя происходит удаление всех его счетов и операций по ним;
При удалении счета происходит удаление всех операций по нему;
---------------
Таблица в MySql: web_crud;
Создание таблиц происходит при запуске;
Для заполнения базы нужно использовать скрипт: data.sql;
